Qs1. Write a JavaScript program to delete all occurrences of element num in a given array. Example: If arr = [1, 2, 3, 4, 5, 6, 2, 3] and num = 2, Result should be: arr = [1, 3, 4, 5, 6, 3]

Qs2. Write a JavaScript program to find the number of digits in a number. Example: If number = 287152, then count = 6

Qs3. Write a JavaScript program to find the sum of digits in a number. Example: If number = 287152, then sum = 25

Qs4. Write a JavaScript program to print the factorial of a number n. Note: Factorial of n is the product of all positive integers less than or equal to n. Example: 7! = 1×2×3×4×5×6×7 = 5040 , 5! = 120 , 3! = 6 , 0! = 1 (always)
